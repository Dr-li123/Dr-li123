Markdown高级技巧

  • 支持的HTML元素
    目前支持的HTML元素有<kbd><b><i><em><sup><sub><br>等
  • 转义
    Markdown 使用了很多特殊符号来表示特定的意义，如果使用显示指定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符
**文本加粗**
\*\*正常显示星号\*\*

输出结果为：
文本加粗
**正常显示星号**

Markdown 支持以下这些符号在前面加反斜杠来帮助插入符号:

\ 反斜线
` 反引号
* 星号
_ 下划线
{} 花括号
[] 方括号
() 小括号
# 井字号
+ 加号
- 减号
. 英文句号
! 感叹号
